Egyéb OmegaT-hez kapcsolódó források

Dokumentáció

Az OmegaT részeként átfogó felhasználói kézikönyv és egy rövid "Gyorskalauz" című áttekintés áll több nyelven rendelkezésére. A dokumentációt megtekintheti önmagában is az Interneten itt.

Segédprogramok

"Aligner" párosító Szerzője Dmitri Gabinski. Pythonban készült.
"Bligner" párosító Szerzője Didier Briel. Pythonban/Perlben készült.
TMXMerger Java parancssorból futtatható kód két vagy több TMX-állomány egyesítésére. Szerzője Henry Pijffers.
TMXCleaner Java parancssorból futtatható kód TMX-állományban lévő "felesleges" fordítási egységek kigyomlálására: kitörli az összes olyan TU-t, amelynek nincs meg a célnyelvi megfelelője, illetve ha annak célnyelvi megfelelője azonos a forrásnyelvivel. Szerzője Henry Pijffers.
Extspell Extspell külső helyesírás-ellenőrző. Szerzője Marc Prior. Futtatásához tcl/tk és Aspell szükséges (mindkettő ingyenes és minden platformon futtatható).
TKount TKount szövegszámláló segédprogram. Szerzője Marc Prior. Futtatásához tcl/tk, Abiword és pdftotext szükséges. Működése Linuxon volt ellenőrizve.
Wordfast TMX-állománykonvertáló A Wordfast által előállított TMX formájú állományokat átalakítja az OmegaT által olvasható formájúra. Szerzője Dmitri Gabinski.
stripIT macro Makró TMX-állományok belső címkéinek eltávolítására. Csak korábbi OmegaT-állományokhoz; az OmegaT jelenleg a TMX-állományokat mind belső címkékkel, mind azok nélkül képes exportálni. Szerzője Sonja Tomaskovic. 
OpenOffice.org szövegdaraboló makrók Segítségükkel az OmegaT a szövegdarabolást bekezdések helyett mondatonként végzi. Csak tájékoztatásul: ez a funkció ma már az OmegaT-ből is elérhető. Szerzője Benjamin Siband.
ReadMe Benjamin Siband szövegdaraboló makróinak leírása.

Egyéb források

Oktató program Oktató program az OmegaT-hez. Része egy adaptálható tanulói kézikönyv tanári kézikönyvvel, továbbá különféle minták.
Ikonok Különféle alternatív ikonok az OmegaT-vel való használatra.
Mintaszövegek Az OmegaT ellenőrzésére és bemutatására.

Lokalizációs források

Ez a portál A portál állományai összecsomagolva az egyéb nyelvi változatot elkészíteni szándékozók számára.
